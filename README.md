# Kids Games React Project

## Project Description
This project is a ReactJS based application that contains multiple small programs and exercises related to JavaScript, React components, and logic building. The project demonstrates the use of React concepts such as functional components, class components, props, state management, conditional rendering, and event handling.

The repository contains different exercises implemented using ReactJS and JavaScript.

---

## Technologies Used
- ReactJS
- JavaScript
- HTML
- CSS
- Node.js

## Exercise Files

### Question 1 – Salary, Grade and Bonus Calculation
- **q1a.jsx**
  - Calculates salary components such as DA, HRA and SA based on basic salary.

- **q1b.jsx**
  - Determines employee grade based on salary range.

- **q1c.jsx**
  - Calculates bonus percentage depending on employee grade.
